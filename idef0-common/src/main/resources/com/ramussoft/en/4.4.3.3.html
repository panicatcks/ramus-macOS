4.4.3.3. Table attributes

Column count is an attribute to set a column count in the table. If this attribute is empty, then the table contains one column.

Query is an attribute to set a query for getting data from a project. A query sets a way of data getting from a project. The query is a set of keywords, divided by dot “.”. First keyword in the query is always a name of base qualifier for the report (see chapter 4.4.3.1. Report attributes). You should perceive every keyword in a query as a qualifier of some type (qualifier of activities, qualifier of streams or any other qualifier created by an user). Depending on this qualifier type, you can use different next keywords. Every keyword also results a qualifier of some type.
