4. Workspaces

It is recommended to work with different static windows of a project in different workspaces. For example, to work with qualifiers in Qualifiers workspaces, to work with diagrams in Diagrams workspace.

But nothing prevent you to work with any windows in any workspace.

A workspace if used to gather relative windows. On the center on main window you can find a place where different dynamic windows are located. Dynamic windows are not available from Windows menu and can be opened with different commands (see below). The same dynamic windows are present on any workspace.
