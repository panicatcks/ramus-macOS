2.5. Tools menu

“Open current project in a web browser” is a command to start a web browser, where you can work with a web interface of the application. How to work with a project in a web browser will be explained below.

“Add Diagram Template” is a command to save a current diagram as a template, which can be used later to create a new diagram with similar structure. The command opens a dialog where you should set a name of the template.

“Diagram Templates” is a command to open a dialog where you can edit already saved templates.

“Project preferences” is a command to open a dialog where you an set different preferences of a current project. This dialog contains preferences:

On a tab “Spelling” you can enable or disable spell checking and set a language of a project.

On a tab “Auto add attributes” you can set a list of attributes, which will be present in just created qualifiers. On this tab you can also set a text attribute, which will be used as a default name of an element.

On a tab “Owner qualifiers” you can find a list of qualifiers, which can be used as owners of processes.

“Application preferences” is a command to open a dialog, where you can set different preferences of the application. The dialog has tabs.

There are three check boxes on a tab “Main preferences”.

If a box “Save application in the memory” is checked, then application will be started much more faster second and next times throw one session of an operation system, but the application will take some memory all the time, even if no files are open.

If a box “Show startup launcher” is checked, then, just before a start of the application, a dialog will be shown where you can setup an automatic open of some file, or open last opened files.

If a box “Check for updates” is checked, then sometimes the application will connect to the site http://ramussoft.co.cc/ and will check for the application's updates. If a new update is available, then a dialog with update message will be shown. The box is not present in corporate and in client versions of the application.

On a tab “Model view properties” you can set a default look of IDEF0 and DFD objects (colors and fonts).
