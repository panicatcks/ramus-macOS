2.1. File menu

File menu contains commands, many of these commands can be executed with a tool bar too.

“New project” is a command to create a new project. This command is available on the tool bar, too.

“Open” is a command to create an existing project. This command is available on the tool bar, too.

“Save” is a command to save current project.

“Save As” is a command to save current project and specify a name of the file where you want to store project.

“Open in new window” is a command to open a new window, where you can work with current project. The command can be useful if you work with two monitors, for example. There is no File menu, in a new window.

“Exit” is a command to close current project. If the project is changed, Ramus will ask you for saving changes to a file.
