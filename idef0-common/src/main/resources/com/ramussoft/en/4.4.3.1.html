4.4.3.1. Report attributes

Base qualifier is an attribute to set a qualifier. The qualifier will be used to build a report. If you build a report for a model, you should select the model for this attribute. This attribute cannot be empty.

Language of keywords is an attribute to set a language of keywords. This attribute cannot be empty.

Model is an attribute to set a model, which is used to build a report. Keywords use this model (see below).
